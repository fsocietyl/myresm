# Sadek Elzakzouk - Portfolio Website

A modern, responsive portfolio website built with pure HTML and CSS (no JavaScript).

## Features

- **Pure HTML/CSS**: No JavaScript required
- **Responsive Design**: Works on all devices
- **Modern UI**: Clean, professional design with animations
- **Side Navigation**: Fixed vertical navigation menu
- **Contact Form**: Direct email integration via mailto links

## Pages

- **Home** (`mainPage.html`) - Hero section and skills overview
- **About** (`about.html`) - Detailed about me and skills
- **Resume** (`resume.html`) - Professional resume with timeline
- **Projects** (`projeler.html`) - Project showcase
- **Contact** (`contact.html`) - Contact information and email form

## Technologies Used

- HTML5
- CSS3 (with animations and modern features)
- Font Awesome Icons
- No JavaScript (pure CSS animations)

## Deployment

This website is designed to be deployed on GitHub Pages. Simply push this repository to GitHub and enable GitHub Pages in the repository settings.
